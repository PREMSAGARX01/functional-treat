# functional-treat
# Data Analyzer and Transformer Program

A Python menu-driven project that demonstrates:

- built-in functions
- user-defined functions
- `*args`
- `**kwargs`
- `__doc__`
- recursion
- lambda functions
- global keyword
- returning multiple values
- 1D and 2D lists
- sorting

## Features

- Input 1D data
- Input 2D data
- Display summary using built-in functions
- Calculate factorial using recursion
- Filter values using lambda
- Sort 1D and 2D data
- Display dataset statistics
- Show function descriptions

## How to Run

```bash
python functional_treat.py

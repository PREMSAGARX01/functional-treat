Data Analyzer and Transformer Program

Objective

This project demonstrates the use of Python functions and data manipulation techniques on one-dimensional (1D) and two-dimensional (2D) arrays using lists.

The project includes:

- Built-in functions
- User-defined functions
- *args and **kwargs
- doc
- Recursion
- Lambda functions
- Global variables
- Returning multiple values
- Sorting collections
- Menu-driven programming

Features

Built-in Functions

- Length of dataset
- Sum of dataset
- Minimum value
- Maximum value
- Average value

User Defined Functions

- Calculate average
- Find duplicate values
- Display unique values

*args

- Accept multiple values dynamically

**kwargs

- Display dataset characteristics using key-value pairs

Recursion

- Factorial calculation

Lambda Functions

- Data filtering
- Data transformation using map()

Global Variable

- Store dataset summary

Return Multiple Values

- Return minimum, maximum and average values

1D and 2D Lists

- User input support
- Display 2D grid format

Sorting

- sort()
- sorted()

Requirements

Python 3.x

How To Run

1. Download the project file.
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Run:

python functional_treat.py
